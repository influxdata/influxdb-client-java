fun create(url: String, username: String, password: CharArray): InfluxDBClientKotlin(source)

Create an instance of the InfluxDB 2.x reactive client.

The username/password auth is based on HTTP "Basic" authentication. The authorization expires when the time-to-live (TTL) (default 60 minutes) is reached and client produces {@link com.influxdb.exceptions.UnauthorizedException}.

The url could be a connection string with various configurations. For more info see: {@link InfluxDBClientOptions.Builder#connectionString(String)}.

Return

client

Parameters

url

the url to connect to InfluxDB (required). Example: http://localhost:8086?readTimeout=5000

username

the username to use in the basic auth

password

the password to use in the basic auth
