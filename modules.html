Project Modules

This project has declared the following modules:

Name Description
The InfluxDB Client Test Tools The InfluxDB client's Test Tools - mocking InfluxDB, parsing HTTP responses
The InfluxDB Client Core InfluxDB Client Core - exceptions, validations, REST client
The InfluxDB 1.7+ Flux Client The client that allow perform Flux Query against the InfluxDB 1.7+.
The Java InfluxDB 2.0 Client The reference Java client that allows query, write and management (bucket, organization, users) for the InfluxDB 2.0.
The RxJava InfluxDB 2.0 Client The reference Java client that allows query and write for the InfluxDB 2.0 by a reactive way.
The Kotlin InfluxDB 2.0 Client The reference Kotlin client that allows query and write for the InfluxDB 2.0 by Kotlin Channel coroutines.
The Scala InfluxDB 2.0 Client The reference Scala client that allows query and write for the InfluxDB 2.0 by Akka Streams.
The Flux DSL Specify the Flux query by the Query Builder.
